markdown 基本语法教程#

空格与换行#

markdown 语法对于空格与换行并不敏感。您即使输入多个空格，markdown 也只会渲染一个空格。例如：

这是一个   示例文本，含有多个空格。

效果：

这是一个 示例文本，含有多个空格。

类似地，多个换行也只会在实际效果中呈现一个换行，例如：

这是第一行。


这是第二行。

效果：

这是第一行。
这是第二行。

如果您想使用多个换行，请使用<br>标签。一个<br>标签代表一次换行，2个<br>标签的效果为2行字中间隔了1行，3个<br>标签的效果为3行字中间隔了2行，以此类推。例如：

这是第一行<br><br><br>这是第二行

效果：

这是第一行


这是第二行

加粗与斜体#

使用**文本**或<strong>文本</strong>来表示加粗文本，使用*文本*或<em>文本</em>来表示斜体文本。我们不推荐使用星号*，而推荐使用strong或em标签，因为前者在某些情况下可能失效，而后者不会失效。例如：

**加粗**
*斜体*
<strong>加粗</strong>
<em>斜体</em>

效果：
加粗
斜体
加粗
斜体

多级标题#

使用#符号来表示标题。一级标题使用一个#，二级标题使用两个#，以此类推。例如：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

效果：

一级标题#

二级标题#

三级标题#

四级标题#

五级标题#
六级标题#

分割线#

使用三个或更多的-、*或_符号可以创建分割线。例如：

---
***
___

效果：

引用块#

使用>符号表示引用块。例如：

> 这是一个引用块。
>> 这是一个嵌套的引用块。

效果：

这是一个引用块。

这是一个嵌套的引用块。

链接#

使用[显示文本](URL)格式创建链接。例如：

[NEU小站](https://www.东北大学.com)

效果：

NEU小站

行内代码#

使用反引号`符号将文字包裹起来，表示行内代码。例如：

这是一个`行内代码`示例。

效果：

这是一个行内代码示例。

代码块#

使用三个反引号```包裹代码块，或者用缩进四个空格来表示代码块。例如：

```
这是一个代码块。
```

效果：

这是一个代码块。

图片#

使用 HTML 标签的 width 和 height 属性来在引用的同时调整图片的大小。例如：

<img src="https://download.xn--xhq44jb2fzpc.com/images/neucartoon.png" alt="图片" width="100" height="100">

效果：

您也可以只设置 width 或 height，浏览器会自动按比例调整图片。例如：

<img src="https://download.xn--xhq44jb2fzpc.com/images/neucartoon.png" alt="图片" width="150">

效果：

表格#

Markdown 允许使用简单的语法创建表格。使用竖线（|）分隔列，使用短横线（-）分隔表头和表内容。以下是一个基本的表格示例：

| 姓名   | 年龄 | 性别 |
| ------ | ---- | ---- |
| 张三   | 25   | 男   |
| 李四   | 30   | 女   |
| 王五   | 28   | 男   |

效果：

姓名 年龄 性别
张三 25 男
李四 30 女
王五 28 男

对齐方式#

通过在分隔表头和表内容的短横线两侧添加冒号（:）可以控制文本对齐方式。可以选择左对齐、右对齐和居中对齐。

  • 左对齐：在短横线左侧添加冒号 :---
  • 右对齐：在短横线右侧添加冒号 ---:
  • 居中对齐：在短横线两侧都添加冒号 :---:

示例如下：

| 姓名   | 年龄 | 性别 |
| :----  | ---: | :--: |
| 张三   | 25   | 男   |
| 李四   | 30   | 女   |
| 王五   | 28   | 男   |

效果：

姓名 年龄 性别
张三 25 男
李四 30 女
王五 28 男
